### 项目结构
```shell
├── biz
├── config
├── dal
├── idl
├── log
├── pkg
├── service
├── static
└── types
```
- biz 路由和api接口
- config 配置文件
- dal 数据库操作层
- idl 接口文件
- pkg 自定义的包，主要存放错误码和工具
- service 服务层
- static 本地文件
- types 返回结构定义
